Detailed Description

Class to encapsulate Servo Signal Input functionality.

This class provides a way to read and decode a Servo signal.

Author
Daniel van den Ouden
Date
Feb-2012
Warning
This class should NOT be used together with the standard Arduino Servo library, use rc::ServoOut instead.
