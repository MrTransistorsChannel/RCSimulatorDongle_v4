Detailed Description

Class to encapsulate PPM Output functionality.

This class provides a way to generate a PPM signal for a configurable amount of channels.

Author
Daniel van den Ouden
Date
Feb-2012
Warning
This class should NOT be used together with the standard Arduino Servo library, use rc::ServoOut instead.
Copyright
Public Domain.

)

Sets up timers and interrupts.

Parameters
p_pinPin to use as output pin, pins 9 and 10 are preferred and give the best result.
p_invertInvert the signal on true.
Note
If precise timing is of importance then you should use either pin 9 or 10, these can be toggled by the timer hardware and will give the best results.
