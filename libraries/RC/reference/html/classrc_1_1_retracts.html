Detailed Description

Class to encapsulate retractable landing gear functionality.

This class provides controls for doors and landing gear.

Author
Daniel van den Ouden

void rc::Retracts::setDelay ( int16_t  p_delay)

Sets the delay between the doors and the gear.

Parameters
p_delayThe delay in milliseconds, range [-10000 - 10000] (-10 - 10 sec).
Note
A negative delay means the gear will lower before the doors have opened completely. This can only be used in combination with a dual servo setup. The library will ensure that the landing gear will not be lowered completely before the doors are completely open and vice versa.
