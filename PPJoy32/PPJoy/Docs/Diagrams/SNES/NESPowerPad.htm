Notes

  • The PowerPad interface was developed by Juan Pablo Sousa. The PPJoy scanning code is also based on his original code for Linux.
  • This interface supports an additional pad on pins 12 and 13.
  • General notes about interface construction
  • This interface supports configurable timing parameters - see Changing timing parameters on how to change these values and SNES/NES timing parameters for their meanings.
