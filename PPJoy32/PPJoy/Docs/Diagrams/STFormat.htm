Notes

  • It is not recommended to build this interface. Rather build Amiga4Player. This interface can be turned into the Amiga4Player interface by moving the Joystick 1 pin 6 (fire) connection from pin 1 (nStrobe) to pin 13 (Select) on the Parallel Port.
  • It is doubtful if this interface will properly work on any PC parallel port - the fire button on joystick 1 will not function properly
