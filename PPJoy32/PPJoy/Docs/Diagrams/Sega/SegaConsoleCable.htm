ConsoleCable Sega interface

Notes

  • This is a pretty obscure interface! It comes from from a old DOS based driver called Console Cable.
  • The +5v can be sourced from the keyboard or joystick port.
  • This interface supports configurable timing parameters - see Changing timing parameters on how to change these values and Sega Genesis controller timing parameters for their meanings.
