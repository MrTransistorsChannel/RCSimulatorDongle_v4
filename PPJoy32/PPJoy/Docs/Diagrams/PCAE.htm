Notes

  • The resistors connected to the parallel port (drawn horizontally) are 1k Ohm
    The resistors connected to pin 1 of the joystick port (drawn vertically) are 4.7k Ohm
  • It is not recommended to build this interface. Rather build Amiga4Player.
  • This interface is not currently supported by PPJoy
